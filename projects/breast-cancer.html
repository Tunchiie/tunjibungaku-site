Project Overview

Built a logistic regression classifier from scratch using only Python and NumPy, achieving 95% test accuracy on breast cancer diagnosis. This project demonstrates deep understanding of machine learning fundamentals by implementing the mathematical foundations without relying on high-level ML libraries.
