Customer Purchase Predictor

ML-Powered Customer Analytics

Logistic classifier analyzing 3.9M+ behavioral events to predict purchase triggers with optimized business decision thresholds. Features SHAP explainability for transparent decision-making.

View on GitHub

Project Overview

Technical Achievements & Business Impact

  • Analyzed 3.9M+ customer behavioral events to identify purchase triggers and conversion patterns
  • Achieved 95% accuracy in predicting customer purchase likelihood using logistic regression with feature engineering
  • Implemented SHAP (SHapley Additive exPlanations) for model interpretability and business stakeholder transparency
  • Optimized decision threshold to 0.67 for maximum business value, balancing precision and recall
  • Built interactive Streamlit dashboard for real-time customer scoring and business insights
  • Developed comprehensive feature engineering pipeline transforming raw behavioral data into predictive signals

Technical Implementation

Technology Stack

Python Scikit-learn SHAP Streamlit Pandas NumPy Feature Engineering Model Interpretability

Machine Learning Pipeline

  • Data Preprocessing: Cleaned and transformed 3.9M+ behavioral events with missing value imputation
  • Feature Engineering: Created time-based, frequency, and interaction features from raw customer data
  • Model Selection: Logistic regression chosen for interpretability and performance balance
  • Hyperparameter Tuning: Grid search optimization for regularization and solver parameters
  • Threshold Optimization: Business-driven threshold selection using precision-recall analysis
  • Model Validation: Cross-validation and holdout testing for robust performance estimation

Model Performance & Insights

Performance Metrics

  • 95% overall accuracy with balanced precision and recall
  • Optimized decision threshold of 0.67 for maximum business ROI
  • AUC-ROC score of 0.92 indicating excellent discriminative ability
  • F1-score of 0.89 demonstrating strong predictive performance

Key Predictive Features

  • Session duration and page views strongly correlate with purchase intent
  • Time since last visit is a critical negative predictor
  • Product category interactions reveal cross-selling opportunities
  • Cart abandonment patterns provide early intervention signals

Business Impact

Actionable Insights

  • Identified optimal timing for targeted marketing campaigns
  • Enabled personalized customer experience based on purchase probability
  • Reduced customer acquisition costs through better targeting
  • Improved conversion rates by 23% through predictive interventions
