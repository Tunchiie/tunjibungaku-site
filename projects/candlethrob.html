CandleThrob

Advanced Algorithmic Trading System

Building an algorithmic trading system that tracks 510+ securities with 24+ years of historical data. Calculates 113+ technical indicators and stores everything in Oracle. Uses real math to analyze market patterns and generate trading signals.

View on GitHub

Project Overview

Technical Achievements & Business Impact

  • Engineered enterprise-scale data pipeline processing 510+ securities with 50M+ data points and 24+ years of historical coverage
  • Implemented 113+ technical indicators and 54 candlestick pattern recognition algorithms using TA-Lib for comprehensive market analysis
  • Designed high-performance Oracle database architecture with incremental loading, bulk operations, and real-time data synchronization
  • Built dual-source integration: Polygon.io for minute-level real-time data and Yahoo Finance for extensive historical analysis
  • Developed robust ETL pipeline with error handling, data validation, and automated testing suite for production reliability
  • Created scalable foundation for neural network and ensemble trading strategies with measurable backtesting capabilities

Technical Implementation

Technology Stack

Python Oracle Database TA-Lib Polygon.io API Yahoo Finance Pandas NumPy SQLAlchemy

System Architecture

  • Data Ingestion Layer: Real-time and historical data collection from multiple sources
  • Processing Engine: Technical indicator calculation and pattern recognition
  • Database Layer: Oracle-based storage with optimized indexing and partitioning
  • Analysis Module: Statistical analysis and signal generation algorithms
  • Backtesting Framework: Historical strategy validation and performance metrics
  • Risk Management: Position sizing and portfolio optimization algorithms
